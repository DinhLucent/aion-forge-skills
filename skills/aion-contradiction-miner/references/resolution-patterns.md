# Resolution Patterns

Use these patterns to resolve contradictions.

- Separate by time: realtime triage now, audited review later.
- Separate by role: operator action, supervisor approval, admin configuration.
- Separate by risk: low-risk automation, high-risk confirmation.
- Progressive disclosure: summary, filter, drilldown, export.
- Shadow mode: observe recommendations before acting on them.
- Human-in-loop: machine prepares, human confirms, system logs.
- Reversible actions: undo, rollback, versioned configuration.
- Explicit uncertainty: confidence labels, missing-data states, escalation.
- Bounded customization: templates plus controlled overrides.
- Event sourcing: preserve auditability while supporting derived views.
